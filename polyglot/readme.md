# Polyglot について

## polyglot-ruby-bash-quine.rb

Ruby兼BashのQuineであり、ソースコードが回文なもの

bash:
```
$ bash -i polyglot-ruby-bash-quine.rb
```

ruby:
```
$ ruby polyglot-ruby-bash-quine.rb
```

テスト方法:
```
$ make 
```
